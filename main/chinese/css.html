# 使用 less

less 是一门向后兼容的 CSS 扩展语言，详情可以查看中文官网 (opens new window)

# 安装和配置

yarn add less less-loader -D
或
npm i less less-loader -D
1
2
3

然后找到根目录的两个 webpack 配置文件 webpack.dev.config 和 webpack.prod.config, 在 module.rules 下新增一段代码

{
    test: /\.less$/i,
    use: [...commonCssLoader, 'less-loader'],
}
1
2
3
4

注意 commonCssLoader 是单独提取出来定义的用来复用的 loader

# 修改 css 文件

现在可以把 css 文件的后缀修改成 less 了

  1. 找到默认目录 /src/css 修改 index.css 文件后缀成 index.less
  2. 修改 /src/index/index.js 文件进行修改
import '../css/index.css'
改成
import '../css/index.less'
