# 项目依赖

Git (opens new window)

Node.js v12+ (opens new window) （注意 windows7 能安装的最高 node 版本是 v13.14.0）

可以使用 npm，但是推荐使用 Yarn

Yarn (opens new window)

# 安装运行

打开仓库 (opens new window)下载项目到本地

git init
git clone https://github.com/wufeng110226/multipage-webpack5-framework.git
cd multipage-webpack5-framework
或网页下载ZIP压缩包，解压至自选目录
