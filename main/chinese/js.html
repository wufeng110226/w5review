# 使用 vue

这是一个 javascript 渐进式框架，易学易用，性能出色，适用场景丰富的 Web 前端框架

更多内容自行查看中文官方 (opens new window)

# 安装

在写这篇教程的时候，我们使用的版本是 Vue 的 v3 版本 v3.2.37

yarn add vue -S
或
npm i vue -S

# 修改 JS 文件

在使用 vue 的特性前，我们修改一下文件

找到/src 目录下的业务 html 文件

向 body 中加入一个 id

<body id="body">
    ...
</body>
1
2
3

找到/src 目录下的业务 js 文件

将下面代码贴入

import { createApp } from 'vue/dist/vue.esm-bundler.js'

const App = {
    data() {
        return {}
    },
    methods: {},
    mounted() {},
}

const app = createApp(App)
app.mount('#body')

# axios

Axios 是一个基于 promise 的网络请求库，可以用于浏览器和 node.js，使用简单，包尺寸小且提供了易于扩展的接口。

详情见中文官网 (opens new window)

先安装

yarn add axios -S
或
npm i axios -S
1
2
3
//找到/src 目录下的业务 js 文件在头部引用
import axios from 'axios'

// 发起一个post请求
axios({
    method: 'post',
    url: '/user/12345',
    data: {
        firstName: 'Fred',
        lastName: 'Flintstone',
    },
})
